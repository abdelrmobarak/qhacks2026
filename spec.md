We are creating an application titled SaturdAI, it includes these features:
-Read emails and offers to reply to emails which it deems necessary
-TLDR Newsletter emails in your inbox
-Tracks subscriptions and bills based off of emails and notifies of renewals
-Weekly To Do List, pulled automatically from user's email
-Default means of comunication with SaturdAI is voice (Text available)
-Google Calendar integration for calendar visualization


Frontend:
React, Tailwind, Electron 

Backend:
Gmail API
Google Calendar API
OpenRouter API (Cheapest model)
Gradium

Bonus Features:
-Agent Mode (text to speech, and speech to text included)
-Call Summarizer
-Summarizing meetings/conversations Microsoft Teams
