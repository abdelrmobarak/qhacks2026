We are creating an application titled SaturdAI, it includes these features:
-Read emails and gives offers on what to reply to emails with
-TLDR Newsletter of the morning news, emails in your inbox, weather
-Tracks subscriptions and bills based off of emails and notifies of renewals
-Weekly To Do list  automated from emails
-Google Calenendar
-Text UI

Frontend:
React, Tailwind, Electron, ShadCN

Backend:
Gmail API
Google Calendar API
OpenRouter API (Cheapest model)
Gradium

Bonus Features:
-Agent Mode (text to speech, and speech to text included)
